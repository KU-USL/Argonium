# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog.

## [1.0.0] - 2025-10-08
### Added
- Initial release of the package.
- 10 scripts included
- AccessTokenManager: Handles the access token for Argon
- APIEnv: Contains environment variables for the API requests
- ArgonAPI: Contains the coroutines for API requests
- CoroutineUtility: Singleton class to make coroutine calls public
- FlightDeclaration: Class for visualizing flight declarations
- FlightDeclarationManager: Class for handling flight declaration objects in the scene
- Geofence: Class for visualizing geofences
- GeofenceManager: Class for handling geofence objects in the scene
- RID: Class for visualizing RID positions
- RIDManager: Class for handling RID objects in the scene
